## Quick Setup

1. Add `nuxt-ripple` dependency to your project

```bash
pnpm add nuxt-ripple
```

2. Add `nuxt-ripple` to the `modules` section of `nuxt.config.ts`

```js
export default defineNuxtConfig({
  modules: [
    'nuxt-ripple'
  ]
})
```
